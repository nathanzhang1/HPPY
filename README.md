# Welcome to HPPY!

HPPY is a fun and friendly app designed to bring positivity to your day. It allows users to share and view motivational messages and images, helping you find a little joy in your daily routine.

## Prerequisites
Before you start, make sure you have the following software installed on your computer:
1. **Node.js**: Download it from [nodejs.org](https://nodejs.org/).
2. **Expo Go App**: Available on the [App Store](https://apps.apple.com/us/app/expo-go/id982107777) or [Google Play Store](https://play.google.com/store/apps/details?id=host.expo.app&hl=en&gl=US).

## Step-by-Step Installation Instructions

1. **Open Terminal**: On your computer, find and open the built-in Terminal application. This can usually be found in the Utilities folder or by searching for "Terminal" in your applications.
   
2. **Install Expo CLI**: Type the following command in the Terminal and hit Enter:
   ```bash
   npm install -g expo-cli
   ```
   This will install Expo CLI, a tool you'll need to run the app.
   
3. **Download the HPPY Project**: Type the following command to clone the HPPY repository:
   ```bash
   git clone https://github.com/nathanzhang1/HPPY.git
   ```
   This downloads the project files to your computer.

4. **Navigate to the Project Folder**: Type the following command and press Enter:
   ```bash
   cd HPPY
   ```
   This takes you into the folder where the project files are located.
 
5. **Install Project Dependencies**: Type this command and press Enter:
   ```bash
   npm install
   ```
   This will install all the necessary libraries that the app needs to run.
   
## How to Run the App

6. **Start the Development Server**: Type the following command and press Enter:
   ```bash
   expo start
   ```
   This will start the app and give you a QR code in the Terminal.

## How to View it on Expo Go

7. **Open Expo Go on Your Mobile Device**: Go to your mobile device and open the Expo Go app.

8. **Scan the QR Code**: Use the Expo Go app to scan the QR code displayed in your Terminal. Once scanned, the app will load on your mobile device.

## Troubleshooting
If you encounter any issues:
- Ensure that you have installed all prerequisites correctly.
- Check your internet connection. A stable connection is required to run the app.
- If the app does not load, try restarting the Terminal and re-running `expo start`.
